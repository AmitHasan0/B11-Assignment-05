Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in dom. Such as getElementById, getElementsByClassName and querySelector, querySelectorAll etc. With these js methods you can select any element by given their class and Id name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

These three may look similar but their uses are in different cases.For example innerHTML is only used for writing multiple line in HTML element. InnerText gets or sets the visible text inside an element. For textContent it gets/sets all the text but also can change the hidden content too.

Question-3: What is event delegation in the DOM?

Event delegation is a process in dom where you add one event listener to the parent element instead of adding separate event listener to every child element. With the help of this process, you can manage every child element as needed.

Question-4: What is event bubbling in the DOM?

Event bubbling in dom is used for targeting an element inside the parent element. This event first works in the target element then bubbles up to the parent element. It means when you click on a button the event also happens for the parent element too. You can stop bubbling by stopPropagation() method.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create element in JavaScript using createElement('tag') method. You just have to write the tag name inside of the method. For adding element you can use appendChild() or append(). You can remove element in JavaScript using remove() or removeChild() method.
